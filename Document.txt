หากคุณใช้ Windows:

คุณสามารถใช้ Task Scheduler บน Windows เพื่อตั้งเวลาการรันสคริปต์ Python ของคุณทุกวัน 1 ครั้ง โดยดำเนินการดังนี้:
1. เปิด Task Scheduler โดยการค้นหา "Task Scheduler" ในเมนู "Start" ของ Windows.
2. คลิกที่ "Create Basic Task" เพื่อสร้างงานใหม่.
3. กำหนดชื่อและคำอธิบายสำหรับงานของคุณ และคลิก "Next".
4. เลือก "Daily" และคลิก "Next".
 5.กำหนดวันที่และเวลาที่คุณต้องการให้งานรัน (เช่น ทุกวันที่ 1 ของเดือน) และคลิก "Next".
6. เลือก "Start a program" และคลิก "Next".
7. ระบุพาธไฟล์ของ Python และสคริปต์ของคุณในช่อง "Program/script" และ "Add arguments (optional)" ตามลำดับ.
8. กด "Next" เพื่อตรวจสอบการตั้งค่าของงาน.
9. กด "Finish" เพื่อสร้างงาน.
นี่คือวิธีที่คุณสามารถตั้งค่าเพื่อให้สคริปต์ Python รันตัวเองทุกวัน 1 ครั้ง โดยใช้ Task Scheduler บนระบบปฏิบัติการของคุณแล้วคุณสามารถปรับเปลี่ยนตามความต้องการในอนาคตได้ตามต้องการ.
